Assignments

Final Project: Complete Portfolio Website

Capstone Project

Objective: Create a comprehensive, professional portfolio website that demonstrates mastery of all course concepts including HTML, CSS, JavaScript, responsive design, and deployment.

Requirements:

  • Minimum 5 pages (Home, About, Projects, Contact, Resume/CV)
  • Fully responsive design working on all device sizes
  • Modern CSS with Flexbox/Grid layout systems
  • Interactive JavaScript features and animations
  • Contact form with validation
  • Project showcase with detailed descriptions
  • Professional typography and color scheme
  • Optimized performance and accessibility
  • SEO best practices implementation
  • Deployed live on GitHub Pages

Technical Specifications:

  • Valid HTML5 semantic markup
  • Modern CSS3 with custom properties
  • Vanilla JavaScript (no frameworks required)
  • Mobile-first responsive design
  • Cross-browser compatibility
  • Fast loading times (< 3 seconds)
  • WCAG 2.1 accessibility standards
  • Clean, well-documented code

Deliverables:

  • Complete website source code
  • Live website URL (GitHub Pages)
  • Project documentation and README
  • Design rationale and process documentation
  • Performance and accessibility audit report
  • Reflection essay on learning journey

Due Date: End of Week 4

Presentation: Final week presentations

Submission: GitHub repository + live URL
