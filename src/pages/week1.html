Learning Objectives

By the end of this week, you will be able to:

  • Apply fundamental design principles (contrast, repetition, alignment, proximity)
  • Understand visual rhetoric and its role in digital communication
  • Create semantic HTML documents with proper structure
  • Set up development environment and version control with Git
  • Deploy websites using GitHub Pages
  • Analyze the rhetorical effectiveness of web design choices

Design & HTML Practice Lab

Lab 1: Design Analysis

Analyze three different websites and identify how they use design principles and visual rhetoric to communicate their message.

Lab 2: Semantic HTML Page

Create your first semantic HTML page with proper structure and meaningful content about yourself or a topic of interest.

Lab 3: Git & GitHub Setup

Set up your development environment and publish your first webpage using Git and GitHub Pages.

Project 1: Visual Design & Rhetoric Analysis

üìÖ Due: Friday, July 11, 2025

Objective:

Create a semantic HTML webpage that demonstrates your understanding of visual design principles and digital rhetoric while establishing your development workflow.

Requirements:

  • Design analysis of 3 different websites (minimum 500 words total)
  • Semantic HTML page with your analysis and reflection
  • Proper document structure using HTML5 semantic elements
  • Accessible markup with appropriate alt text and headings
  • Valid HTML (pass W3C validation)
  • Published on GitHub Pages with repository showing commit history
  • Professional presentation and clear writing

Evaluation Approach:

  • Demonstrate depth and accuracy of design analysis
  • Show proficiency in HTML semantic structure and validation
