Assignment 2: Responsive Portfolio Redesign

Major Project

Due: Saturday, July 20, 2025 by 11:59 PM

Points: 50

Objective: Transform your Week 1 portfolio page into a beautiful, responsive website using modern CSS techniques. Focus on clean design, typography, and mobile-first responsive layout.

Requirements:

  • External CSS stylesheet with organized, commented code
  • Custom color scheme and typography choices
  • Flexbox or CSS Grid layout for main sections
  • Responsive design that works on mobile, tablet, and desktop
  • Navigation bar with hover effects
  • Image optimization and responsive images
  • CSS transitions and subtle animations
  • Valid CSS (use W3C CSS validator)
  • Cross-browser compatibility testing

Design Guidelines:

  • Choose a cohesive color palette (3-5 colors max)
  • Use consistent spacing and typography scale
  • Ensure sufficient color contrast for accessibility
  • Keep design clean and professional
  • Test on actual devices or browser dev tools
