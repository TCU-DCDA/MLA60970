Week 4 Assignments

This final week focuses on portfolio development and professional presentation of your work.

Final Project: Complete Portfolio Website

Capstone Project

Due: Sunday, August 3, 2025 by 11:59 PM

Points: 100

Objective: Create a comprehensive, professional portfolio website that demonstrates mastery of all course concepts including HTML, CSS, JavaScript, responsive design, and deployment.

Portfolio Requirements:

  • Minimum 5 pages (Home, About, Projects, Contact, Resume/CV)
  • Fully responsive design working on all device sizes
  • Modern CSS with Flexbox/Grid layout systems
  • Interactive JavaScript features and animations
  • Contact form with validation
  • Project showcase with detailed descriptions
  • Professional typography and color scheme
  • Optimized performance and accessibility
  • SEO best practices implementation
  • Deployed live on GitHub Pages

Technical Specifications:

  • Valid HTML5 semantic markup
  • Modern CSS3 with custom properties
  • Vanilla JavaScript (no frameworks required)
  • Mobile-first responsive design
  • Cross-browser compatibility
  • Fast loading times (< 3 seconds)
  • WCAG 2.1 accessibility standards
  • Clean, well-documented code

Deliverables:

  • Complete website source code
  • Live website URL (GitHub Pages)
  • Project documentation and README
  • Design rationale and process documentation
  • Performance and accessibility audit report

Self Reflection 3: Final Assessment & Grade Argument

Final Assessment

Due: Sunday, August 3, 2025 by 11:59 PM

Points: 25

Objective: Complete a comprehensive final self-assessment documenting your learning journey, argue for your final course progress, and articulate the value of your learning experience.

Requirements:

  • 1200-1500 word comprehensive reflective essay
  • Document complete learning journey from Week 1 to Week 4
  • Argue for your final course grade with substantial evidence
  • Analyze growth in technical skills (HTML, CSS, JavaScript)
  • Reflect on problem-solving abilities and creative development
  • Evaluate community contributions and peer engagement
  • Connect learning to future personal/professional applications
  • Include portfolio of best work as supporting evidence

Assessment Components:

  • Depth and honesty of self-reflection
  • Quality of evidence and examples provided
  • Growth mindset demonstration
  • Professional presentation and writing quality
  • Realistic and thoughtful self-assessment
