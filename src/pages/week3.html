Assignments

Assignment 3: Interactive Web Application

Major Project

Objective: Create a fully interactive web application that demonstrates mastery of JavaScript fundamentals, DOM manipulation, and event handling.

Requirements:

  • Implement at least 5 different interactive features
  • Use proper JavaScript variable declarations (let, const)
  • Include form validation with user feedback
  • Demonstrate event handling for multiple event types
  • Use DOM manipulation to dynamically update content
  • Include error handling and user input validation
  • Implement local storage for data persistence
  • Write clean, commented JavaScript code

Deliverables:

  • Complete HTML, CSS, and JavaScript files
  • README file explaining features and functionality
  • Screenshots or screen recording of app in action
  • Reflection document on challenges and solutions

Due Date: End of Week 3

Submission: Upload to GitHub repository and submit link
