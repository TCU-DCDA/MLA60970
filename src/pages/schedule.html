Week 1: Visual Design & HTML Foundations

Date Mode Topic Due Dates
M 7/7 Lecture / Discussion
Design Principles & Visual Rhetoric
  • CRAP design principles (Contrast, Repetition, Alignment, Proximity)
  • Visual hierarchy and information architecture
  • Color theory basics and psychological impact
  • Typography fundamentals for web design
T 7/8 Tutorial
HTML Semantic Structure
  • HTML5 semantic elements (header, main, section, article, aside, footer)
  • Document structure and content hierarchy
  • Accessibility considerations in HTML markup
  • Forms, lists, and media elements
W 7/9 Workshop
Development Environment & Git
  • Setting up VS Code for web development
  • Git version control fundamentals
  • Creating and managing GitHub repositories
  • Deploying websites with GitHub Pages
R 7/10 Hands-on Lab
  • Design Analysis Lab
  • Semantic HTML Page Creation
  • Git & GitHub Setup
F 7/11 Assignment Due
Week 1 Assignment: Visual Design Analysis & HTML Foundation
  • Design analysis of 3 different websites (minimum 500 words total)
  • Semantic HTML page with analysis and reflection
  • Published on GitHub Pages

Self Reflection 1: Learning Foundations due Saturday 7/12 by 11:59 PM

Week 2: CSS Styling & Layout

Date Mode Topic Due Dates
M 7/14 Lecture / Tutorial
CSS Fundamentals & Selectors
  • CSS syntax and how it connects to HTML
  • Types of selectors (element, class, ID, descendant)
  • The CSS box model (margin, border, padding, content)
  • Color systems and typography basics
T 7/15 Workshop
Flexbox Layout Mastery
  • Flexbox concepts: flex container vs flex items
  • Main axis and cross axis understanding
  • Essential flexbox properties
  • Building common layout patterns (navbar, card grids, centering)
W 7/16 Lecture / Workshop
Responsive Design & Media Queries
  • Mobile-first design philosophy and why it matters
  • CSS media queries syntax and breakpoints
  • Responsive units (%, em, rem, vh, vw)
  • Making images and media responsive
R 7/17 Hands-on Lab
  • Style Your HTML Page Lab
  • Flexbox Navigation Lab
  • Responsive Card Layout Lab
F 7/18 Project Work
Portfolio Enhancement & Testing
  • Transform Week 1 portfolio with CSS
  • Cross-browser compatibility testing
  • Responsive design testing
M 7/21 Assignment Due
Week 2 Assignment: Responsive Portfolio Redesign
  • External CSS stylesheet with organized, commented code
  • Custom color scheme and typography choices
  • Flexbox or CSS Grid layout for main sections
  • Responsive design that works on mobile, tablet, and desktop
Responsive Portfolio Redesign due by 11:59 PM
Self Reflection 2: Midterm Progress & Grade Argument due Tuesday 7/22 by 11:59 PM

Week 3: JavaScript Fundamentals

Date Mode Topic Due Dates
T 7/22 Lecture / Tutorial
JavaScript Basics & Syntax
  • JavaScript fundamentals including variables, data types, operators
  • Control structures and functions
  • Working with the browser console
  • Debugging basics
W 7/23 Workshop
DOM Manipulation & Events
  • Understanding the Document Object Model (DOM)
  • Selecting elements and modifying content
  • Event handling for user interactions
  • Creating dynamic web features
R 7/24 Workshop
Debugging with Developer Tools
  • Using browser developer tools effectively
  • JavaScript console for debugging

  • Performance monitoring basics
F 7/25 Hands-on Lab
  • Interactive Form Builder Lab
  • Image Gallery Lab
  • DOM Inspector Tool Lab

Week 3 Assignment: Interactive Web Application
  • Implement at least 5 different interactive features

Interactive Web Application due by 11:59 PM
