Course Assignments

This course features four major assignments, each building upon the previous week's learning objectives. All assignments emphasize practical application, creative problem-solving, and professional development in web authoring.

  • Analyze your experience with visual design principles
  • Reflect on your HTML learning progress and challenges
  • Document your development environment setup experience
  • Articulate specific skills gained and areas for growth

Assignment 1: Visual Design Analysis & HTML Foundation

üìÖ Due: Friday, July 11, 2025 by 11:59 PM

Objective:

Create a semantic HTML webpage that demonstrates your understanding of visual design principles and digital rhetoric while establishing your development workflow.

Requirements:

  • Design analysis of 3 different websites (minimum 500 words total)
  • Semantic HTML page with your analysis and reflection
  • Identify CRAP design principles (Contrast, Repetition, Alignment, Proximity)
  • Analyze visual hierarchy and information architecture
  • Proper document structure using HTML5 semantic elements
  • Accessible markup with appropriate alt text and headings
  • Valid HTML (pass W3C validation)
  • Published on GitHub Pages with repository showing commit history
  • Professional presentation and clear writing

Evaluation Approach:

  • Demonstrate depth and accuracy of design analysis
  • Show proficiency in HTML semantic structure and validation
  • Present clear, well-organized writing
  • Evidence proper Git workflow and GitHub Pages deployment

Skills Developed:

Visual Design Analysis HTML5 Semantic Markup Git Version Control GitHub Pages Deployment Digital Rhetoric

Assignment 2: Responsive Portfolio Redesign

üìÖ Due: Monday, July 21, 2025 by 11:59 PM

Objective:

Transform your Week 1 portfolio page into a beautiful, responsive website using modern CSS techniques. Focus on clean design, typography, and mobile-first responsive layout.

Requirements:

  • External CSS stylesheet with organized, commented code
  • Custom color scheme and typography choices
  • Flexbox or CSS Grid layout for main sections
  • Responsive design that works on mobile, tablet, and desktop
  • Navigation bar with hover effects
  • Image optimization and responsive images
  • CSS transitions and subtle animations
  • Valid CSS (use W3C CSS validator)
  • Cross-browser compatibility testing

Design Guidelines:

  • Choose a cohesive color palette (3-5 colors max)
  • Use consistent spacing and typography scale
  • Ensure sufficient color contrast for accessibility
  • Keep design clean and professional
  • Test on actual devices or browser dev tools

Skills Developed:

CSS3 Fundamentals Flexbox Layout Responsive Design Media Queries Typography Color Theory

Assignment 3: Interactive Web Application

üìÖ Due: Sunday, July 27, 2025 by 11:59 PM üî• Major Project

Objective:

Create a fully interactive web application that demonstrates mastery of JavaScript fundamentals, DOM manipulation, and event handling.

Requirements:

  • Implement at least 5 different interactive features
  • Use proper JavaScript variable declarations (let, const)
  • Include form validation with user feedback
  • Demonstrate event handling for multiple event types
  • Use DOM manipulation to dynamically update content
  • Include error handling and user input validation
  • Implement local storage for data persistence
  • Write clean, commented JavaScript code

Interactive Feature Ideas:

  • JavaScript Calculator with operation history
  • Interactive To-Do List with task management
  • Image Gallery with filtering and search
  • Form Builder with dynamic field creation
  • DOM Inspector Tool for web development

Deliverables:

  • Complete HTML, CSS, and JavaScript files
  • README file explaining features and functionality
  • Screenshots or screen recording of app in action
  • Reflection document on challenges and solutions

Skills Developed:

JavaScript ES6+ DOM Manipulation Event Handling Form Validation Local Storage Debugging

Final Project: Complete Portfolio Website & Self Reflection 3

üìÖ Due: Sunday, August 3, 2025 by 11:59 PM üèÜ Capstone Project

Objective:

Create a comprehensive, professional portfolio website that demonstrates mastery of all course concepts including HTML, CSS, JavaScript, responsive design, and deployment. Additionally, write a reflective essay documenting your learning journey.

Portfolio Requirements:

  • Minimum 5 pages (Home, About, Projects, Contact, Resume/CV)
  • Fully responsive design working on all device sizes
  • Modern CSS with Flexbox/Grid layout systems
  • Interactive JavaScript features and animations
  • Contact form with validation
  • Project showcase with detailed descriptions
  • Professional typography and color scheme
  • Optimized performance and accessibility
  • SEO best practices implementation
  • Deployed live on GitHub Pages

Technical Specifications:

  • Valid HTML5 semantic markup
  • Modern CSS3 with custom properties
  • Vanilla JavaScript (no frameworks required)
  • Mobile-first responsive design
  • Cross-browser compatibility
  • Fast loading times (< 3 seconds)
  • WCAG 2.1 accessibility standards
  • Clean, well-documented code

Deliverables:

  • Complete website source code
  • Live website URL (GitHub Pages)
  • Project documentation and README
  • Design rationale and process documentation
  • Performance and accessibility audit report

Skills Developed:

Portfolio Design Professional Presentation Project Curation Reflective Writing Web Performance Accessibility SEO
