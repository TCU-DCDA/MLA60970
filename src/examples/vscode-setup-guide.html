1Download Visual Studio Code

Visit the official VS Code website and download the installer for your operating system:

  • Windows: Download the .exe installer
  • macOS: Download the .zip file or use Homebrew
  • Linux: Download the .deb, .rpm, or .tar.gz package

4Customize Your Settings

Recommended settings for web development:

  • Auto Save: Enable "onFocusChange" for automatic saving
  • Format on Save: Enable automatic code formatting
  • Tab Size: Set to 2 spaces for HTML/CSS
  • Word Wrap: Enable for better readability

5Test Your Setup

Create a simple HTML file to test your environment:

  1. Create a new file called test.html
  2. Type ! and press Tab to generate HTML boilerplate
  3. Right-click the file and select "Open with Live Server"
  4. Your browser should open showing your page
