HTML Element Reference

Document Structure

<!DOCTYPE html> <html lang="en"> <head> <title>Page Title</title> </head> <body> Content here </body> </html>

Semantic Elements

  • <header> - Page or section header
  • <nav> - Navigation links
  • <main> - Main content area
  • <section> - Thematic content section
  • <article> - Independent content
  • <aside> - Sidebar content
  • <footer> - Page or section footer

Common Elements

  • <h1> to <h6> - Headings (importance order)
  • <p> - Paragraph text
  • <a href=""> - Links
  • <img src="" alt=""> - Images
  • <ul>/<ol> - Lists
  • <div> - Generic container
  • <span> - Inline container

Practice Exercise

Try to identify the purpose of each element before clicking. Understanding semantic HTML helps with accessibility and SEO!

üí° Why Semantic HTML Matters:

  • Accessibility: Screen readers understand your content structure
  • SEO: Search engines better understand your page content
  • Maintainability: Code is easier to read and modify
  • Styling: CSS can target semantic elements more effectively
