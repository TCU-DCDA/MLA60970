Course Description & Core Principles:

This course provides a foundational introduction to web authoring, blending technical skills with critical inquiry into digital culture. Students will explore HTML, CSS, and basic JavaScript to create and structure web content while considering design principles, accessibility, and user experience. Through hands-on projects, students will develop functional and aesthetically engaging websites while examining the broader implications of digital authorship in contemporary society. No prior coding experience is required. By the end of the course, students will have a strong foundation in web design and an understanding of how digital platforms shape communication, creativity, and public discourse.

Core Principles

  • Technical skills with critical inquiry - blending web development with digital culture analysis
  • Hands-on project-based learning - creating functional and aesthetically engaging websites
  • Inclusive design focus - emphasizing accessibility and user experience principles
  • Digital authorship exploration - examining how web platforms shape communication and creativity
  • Beginner-friendly progression - no prior coding experience required, building strong foundations
