Course Overview

This intensive 4-week course introduces students to the essential skills of web authoring. You'll dive into the core technologies that power the modern web while engaging in practical, vibe-based coding exercises that make learning both effective and enjoyable.

What You'll Learn

  • HTML fundamentals and semantic markup
  • CSS styling, layouts, and responsive design
  • JavaScript programming and DOM manipulation
  • Modern web development workflows
  • Best practices for accessible web design

Course Format: Asynchronous online learning with hands-on projects, video tutorials, and peer collaboration opportunities.
