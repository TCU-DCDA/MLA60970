Flow 1

Make it Beautiful

Visual Expression & Creative Foundation

Duration: 1 week Dates: July 7-14, 2025

Learning Invitation

"What would you create if you could make anything beautiful on the web?"

This first flow is all about visual expression and aesthetic discovery. You'll explore the fundamentals of web design not through rigid tutorials, but through creative projects that inspire you to make something beautiful. This is where your journey into structured creativity begins.

Skill Emergence Points

Skills that naturally develop through your creative projects:

HTML Semantic Structure

  • Meaningful content organization and hierarchy
  • Semantic elements for better accessibility
  • Clean, readable code structure
  • SEO-friendly markup foundations

CSS Fundamentals

  • Selectors, properties, and the cascade
  • Box model and layout principles
  • Color values and background techniques
  • CSS specificity and inheritance

Typography & Color Theory

  • Font selection and pairing principles
  • Color harmony and visual hierarchy
  • Contrast and readability optimization
  • Emotional impact through design choices

Responsive Design

  • Mobile-first design thinking
  • Media queries and breakpoints
  • Flexible layouts and grid systems
  • Cross-device testing strategies
